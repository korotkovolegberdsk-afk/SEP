## Назначение папок

SEP.Core
    Общие классы платформы.

SEP.Database
    Работа с SQLite.

SEP.Import
    Импорт данных
        YGX
        Altium
        Gerber
        BOM

SEP.Library
    Бизнес-логика.

SEP.Viewer
    Пользовательский интерфейс WPF.

database
    Файлы SQLite.

docs
    Документация проекта.

drawings
    Схемы и рисунки.

samples
    Примеры файлов
        YGX
        Gerber
        BOM
        CAD

tests
    Тестовые проекты.

build
    BAT-файлы.

Tools
    Вспомогательные программы.

## Перед отправкой архива

1. Выполнить Clean.
2. Проверить сборку.
3. Убедиться, что нет bin и obj.
4. Архивировать проект.

Изменяем только полный файл.

Каждый пакет должен собираться.
